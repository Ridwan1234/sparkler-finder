

## Investment Platform — Build Plan

### Phase 1: Landing Page
- **Header/Navigation**: Logo, nav links (Home, About, Services, Plans, FAQs, Contact), Login & Signup buttons
- **Hero Section**: Dark green gradient background with bold headline ("Smart Solution For Business"), description text, Get Started & Login CTAs
- **About Section**: "Experienced Trading Platform" with 3 feature cards (Proven Track Record, Transparency & Security, Customer Support) + company description
- **Services Grid**: Card grid showcasing services — Forex & Stocks, Crypto Trading, Real Estate, Loaning, NFTs, Mining, Metaverse Investing, Wealth Management, Gold, Crude Oil, Agriculture, ETFs
- **Why Choose Us Section**: Feature highlights with icons (Real-time data, Customer support, Higher security, Lower commissions)
- **Investment Plans Section**: Pricing cards for different tiers (Standard, Silver, Golden, Diamond packs) showing ROI, min/max amounts, duration
- **Testimonials Section**: User reviews carousel
- **FAQ Section**: Accordion-style frequently asked questions
- **Footer**: Company info, quick links, social media icons, newsletter signup

### Phase 2: Authentication (Supabase)
- **Signup Page**: Registration form with email, password, name, referral code
- **Login Page**: Email/password login
- **Password Reset**: Forgot password flow with reset page
- **User Profiles Table**: Store display name, avatar, phone, and other user info

### Phase 3: User Dashboard
- **Dashboard Layout**: Sidebar navigation + main content area
- **Overview Page**: Account balance, active investments, total earnings, recent transactions
- **Investment Plans Page**: Browse and select investment plans, make deposits
- **Transaction History**: Table of all deposits, withdrawals, and earnings
- **Deposit Page**: Choose payment method (crypto wallets), enter amount, submit deposit
- **Withdrawal Page**: Request withdrawal with wallet address input
- **Profile Settings**: Edit personal info, change password, security settings
- **Referral System**: Unique referral link, referral earnings tracking

### Design System
- Dark green primary color (#0a8f4e) with dark backgrounds
- Modern, clean typography
- Card-based layouts with subtle shadows and gradients
- Mobile-responsive design throughout
